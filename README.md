# MediSense- Smart Symptom Checker

## Overview

Smart Symptom Checker is a web-based healthcare application that helps users identify possible diseases based on the symptoms they enter. The system uses a Machine Learning model trained on symptom-disease data to predict the most likely disease and suggest the appropriate medical specialist.

This project was developed as a learning project to understand the practical implementation of Machine Learning, Flask APIs, and frontend-backend integration.

## Features

* Predicts possible diseases from user symptoms
* Recommends relevant medical specialists
* Checks severity level of symptoms
* Interactive and user-friendly interface
* Machine Learning based prediction system
* Flask REST API backend
* Responsive frontend using HTML, CSS, and JavaScript

## How It Works

1. User enters symptoms through the web interface using either voice-based input or keyboard-based text input.
2. The frontend sends the symptoms to the Flask backend.
3. The Machine Learning model processes the symptoms.
4. The system predicts the most likely disease.
5. Severity of the condition is analyzed.
6. A suitable medical specialist is recommended.
7. Results are displayed to the user.

## Installation

### Clone the Repository

```bash
git clone https://github.com/DikshaViswanath/smart_symptom_checker.git
cd smart_symptom_checker
```

### Install Dependencies

```bash
pip install -r backend/requirements.txt
```

### Run the Flask Server

```bash
python backend/app.py
```

### Open the Application

Open the frontend file in your browser:

```text
frontend/index.html
```
